Final Project Proposal

  • Application Name: Mona_Draw
  • Team: Adela
  • Proposal Update: Sprint 5, March 29, 2021
  • Brief Description: The user should be able to input the file name and watch the program run
    on the screen, outputting a likeness of a cat.
  • Prioritized List of Features: input prompt, background and line colors, completed graphics
  • Non-Prioritized List of Features: animation of some sort
  • Languages and Platforms utilized: Python (version3) and Visual Studio Code
  • Technical Features From the Class: color control (color()), drawing state (penup(), pendown()),
    turtle motion (forward(), right(), speed(), goto()), window control (.setup), multi colored output,
    text file utilization
  • Technical Features beyond the Class: two turtles working independentely, implementing more methods,
    drawing circles/concentric circles
